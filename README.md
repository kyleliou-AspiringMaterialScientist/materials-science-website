# Materia — Materials Science Explorer

## Overview
Materia is a website that I have created out of my own ambition and commitment to my own education. I have created this website in order to showcase some of the niche elements of material science that are routinely overlooked within out daily lives.
The project covers fundamental concepts such as atomic structures, chemical bonding, crystal structures, material properties, and manufacturing methods. It also explores important engineering materials including silicon, titanium, aluminum, nickel, graphene, polymers, ceramics, carbon fiber, and lithium-based materials.
This is a short-term passion project that I have created in a timespan of roughly 3 months with many breaks in between.

## Features

- Exploration of important engineering materials
- Applications in smartphones, electric vehicles, aerospace, and medical implants
- Information about future challenges in materials science
- Sections covering energy storage, hydrogen energy, semiconductors, sustainable materials, and biomaterials
- Interactive website design with HTML, CSS, and JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Purpose

This project was created to make materials science more accessible and inspire interest in engineering, chemistry, physics, and advanced technologies.
